# Software Engineering Lab - React Todo App

This project is a modern, feature-rich React Todo application developed for the Software Engineering Lab. It demonstrates best practices in React development, accessibility, internationalization, testing, and code organization.

## Key Features
- Add, edit, delete, and complete todos
- Due dates and overdue notifications
- Drag-and-drop reordering
- Priority system
- Filtering, search, and bulk actions
- LocalStorage persistence
- Inline editing
- Undo/redo actions
- Responsive design & dark mode
- Multi-language (FA/EN)
- User onboarding tour
- Accessibility (ARIA, keyboard navigation)
- Integration tests

## Quick Start

1. **Install dependencies:**
   ```bash
   cd software-engineering-lab
   npm install
   ```
2. **Run the app:**
   ```bash
   npm start
   ```
   The app will be available at [http://localhost:3000](http://localhost:3000).
3. **Run tests:**
   ```bash
   npm test
   ```
